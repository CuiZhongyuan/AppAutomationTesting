# 基于po模型开发

包目录说明
- base包：负责存放公用的操作方法
- page包：负责用于元素的定位
- handle包:获取元素后的操作
- business包:负责业务逻辑用例
- testcase包:负责用例维护
- listener包:报告生成监听器
- utils包:工具类
- execption包：统一异常处理
